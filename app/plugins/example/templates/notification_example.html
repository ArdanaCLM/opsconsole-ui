Notifications

Simple notifications can be displayed at the top of the page with a background status color and a message.

The notification bar provides a way to cycle through the notifications and well as a way to clear a notification. The "X" button on the right side of the notification header clears the current message. The left and right arrows scroll through the notification messages.

The following API call can be used in a controller to add the notification:

       addNotification(status, translation_key);

The message translation key is a defined tag for a localized string, found for each localized language folder in the "locals" folder.

Status:

The following status strings map to a background color in the notification header.

Status Background Color
info green
warn yellow
error red

JavaScript:

     addNotification('info', 'common.welcome');
     addNotification('warn', 'common.test_message');
     addNotification('error', 'common.test_message2');
Example Controller:

    p.controller('ExampleController', ['$scope', 'addNotification',
       function($scope, addNotification) {
         $scope.doSomething = function(){
                  ...
                  addNotification('warn', 'common.welcome');
                  ...
         };
       }
   ]);
