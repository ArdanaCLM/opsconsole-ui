Attributes: 
There is only one attribute to specify for the 'tabbedpage' directive:
- pageList
      a list of objects which contain the following properties to be used for each tab:
      1. header
            name of the tabbed page
      2. template
            URL to the page's template

HTML Code: 
<h3>{{"example.tabbedpage" | translate}}</h3>
<br>

<tabbedpage pageList="myPages"></tabbedpage>

                {header: 'example.tabbedpage.content2', template: 'example/templates/tabbed_page_content2.html'},
