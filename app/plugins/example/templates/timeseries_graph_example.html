Parameters: 
The following are parameters you can use when using the 'timeseries' directive:
- data (required)
      array of graph data in [timestamp, value] format
- config (required)
      graph configuration object which contains the following properties:
        * title (required)
              graph title string
        * type (required)
              graph type string: 'line', 'area', 'stackedArea', 'bar', 'stackedBar'
        * actionMenu (optional, no action menus if not provided)
              object contains action menu config:
                - label   (menu label string)
                - action  (menu function definition)
        * timeUnit (optional, use Flot's label if not provided)
              time unit string for x-axis labels: 'minute', 'hour', 'day'
- wide (optional)
      boolean indicates a wide graph, if not specified the graph size will be regular
